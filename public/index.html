API de Recetas de la Abuela

Bienvenido a la API de Recetas de la Abuela.
Esta API permite registrar usuarios y crear recetas para compartir con familiares y amigos.

Endpoints principales

  • /api/users/register - Registrar usuario (POST)
  • /api/users/login - Iniciar sesión (POST)
  • /api/users/profile - Ver perfil de usuario (GET)
  • /api/recipes - Listar recetas (GET) / Crear receta (POST)
  • /api/recipes/search?q=palabra - Buscar recetas (GET)
  • /api/recipes/:id - Ver receta por ID (GET)
  • /api/recipes/:id - Actualizar receta (PUT)
  • /api/recipes/:id - Eliminar receta (DELETE)
  • /api/groups - Listar grupos (GET) / Crear grupo (POST)
  • /api/groups/:id - Ver grupo por ID (GET)
  • /api/groups/:id - Eliminar grupo (DELETE)
